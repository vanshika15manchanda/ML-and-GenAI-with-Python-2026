## 📝 Project Description
This repository contains the solution for **Assignment 2** of the *6 Weeks Internship on Machine Learning and Generative AI using Python*. The script focuses on implementing fundamental programming concepts such as control flow statements (`if-elif-else`) and loops (`for`).

### 🚀 Implemented Programs:
1. **Sum of First 10 Natural Numbers:** Uses a `for` loop to calculate the cumulative sum.
2. **Factorial of a Number:** Calculates the factorial of a given number using iterative logic.
3. **Fibonacci Series:** Prints the Fibonacci sequence up to a specified number of terms.
4. **Largest Among 3 Numbers:** Evaluates and compares three numbers using `if-elif-else` conditions.
5. **Student Result System:** An interactive program that takes student details and multiple subject marks as input (via loops), calculates the total percentage, and assigns an alphabetical grade.
